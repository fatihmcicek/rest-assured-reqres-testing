# REST Assured ReqRes API Testing Project

This project demonstrates API testing using REST Assured framework with ReqRes API (https://reqres.in/).

## Technologies & Tools
- Java 21
- REST Assured
- TestNG
- Maven
- Lombok
- Jackson
- Log4j

## Test Scenarios
- User Management Tests
    - Get List Users
    - Get Single User
    - Create User
    - Update User
    - Delete User
    - Register User
    - Login User
